# Tasahil Complete Service Icons v2

حزمة SVG كاملة للخدمات المستلمة: **67 خدمة** ضمن **8 تصنيفات**.

- SVG 24×24
- fill: none
- stroke: currentColor
- stroke-width: 1.8
- ملفات منفصلة لكل خدمة
- catalog.json عربي/إنجليزي
- icon-registry.ts جاهز للربط
- preview.html للمعاينة

## الاستخدام البصري

الحجم المقترح للأيقونة 24px داخل حاوية 44×44px، خلفية #FFF1F2 ولون #E31B23 ونصف قطر 12px.

## الربط

احفظ iconKey في الباكند وأعده ضمن Service Catalog DTO. لا تربط الأيقونة باسم الخدمة المترجم. خدمة AIR_FILTER_REPLACEMENT محفوظة enabled=false حسب القائمة المستلمة.
